# Git

## 概要

Git の基本設定メモをまとめる。

## ローカル設定

### ユーザー設定

```bash
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
```

### エディタ設定

```bash
git config --global core.editor "editor name"
```
